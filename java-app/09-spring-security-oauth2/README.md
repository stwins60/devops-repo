# Spring Security OAuth2

OAuth2 authentication with Spring Security.

## Features

- OAuth2 authentication
- JWT tokens
- Role-based access
- Protected endpoints

## Project Structure

```
09-spring-security-oauth2/
├── src/
│   ├── main/
│   │   ├── java/com/example/security/
│   │   │   ├── SecurityApplication.java
│   │   │   ├── config/
│   │   │   └── controller/
│   │   └── resources/
│   └── test/
├── pom.xml
└── README.md
```

## Running

```bash
mvn spring-boot:run
```
