# Spring Batch Application

Batch processing application with Spring Batch.

## Features

- Batch job processing
- Step configuration
- Chunk processing
- Job scheduling
- Item readers/writers

## Project Structure

```
07-spring-batch/
├── src/
│   ├── main/
│   │   ├── java/com/example/batch/
│   │   │   ├── BatchApplication.java
│   │   │   ├── config/
│   │   │   ├── job/
│   │   │   └── processor/
│   │   └── resources/
│   └── test/
├── pom.xml
└── README.md
```

## Running

```bash
mvn spring-boot:run
```
