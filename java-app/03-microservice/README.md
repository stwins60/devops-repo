# Spring Boot Microservice

Microservice with Spring Boot and Spring Cloud.

## Features

- Microservice architecture
- Service discovery
- API Gateway
- Load balancing
- Circuit breaker

## Project Structure

```
03-microservice/
├── src/
│   ├── main/
│   │   ├── java/com/example/microservice/
│   │   │   ├── MicroserviceApplication.java
│   │   │   ├── controller/
│   │   │   ├── service/
│   │   │   └── config/
│   │   └── resources/
│   └── test/
├── pom.xml
└── README.md
```

## Running

```bash
mvn spring-boot:run
```
