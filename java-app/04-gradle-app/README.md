# Gradle Application

Application built with Gradle build tool.

## Features

- Gradle build system
- Multi-module project
- Dependency management
- Custom tasks

## Project Structure

```
04-gradle-app/
├── src/
│   ├── main/
│   │   ├── java/com/example/gradle/
│   │   │   └── Application.java
│   │   └── resources/
│   └── test/
├── build.gradle
├── settings.gradle
└── README.md
```

## Build

```bash
./gradlew build
```

## Running

```bash
./gradlew run
```
