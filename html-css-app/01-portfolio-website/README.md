# Portfolio Website

Modern portfolio website with HTML, CSS, and JavaScript.

## Features

- Responsive design
- Smooth animations
- Contact form
- Projects showcase
- About section

## Project Structure

```
01-portfolio-website/
├── index.html
├── about.html
├── projects.html
├── contact.html
├── css/
│   ├── styles.css
│   ├── responsive.css
│   └── animations.css
├── js/
│   ├── main.js
│   └── form.js
├── images/
└── README.md
```

## Usage

Open `index.html` in a web browser or serve with a local server:

```bash
python -m http.server 8000
# or
npx serve
```

Access at: http://localhost:8000
