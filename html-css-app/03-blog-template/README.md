# Blog Template

Responsive blog template with articles and sidebar.

## Features

- Blog post layout
- Sidebar with categories
- Responsive design
- Search functionality
- Comment section

## Project Structure

```
03-blog-template/
├── index.html
├── post.html
├── css/
│   ├── blog.css
│   └── post.css
├── js/
│   └── search.js
└── README.md
```

## Usage

Open `index.html` to view blog listing.
