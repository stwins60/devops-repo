# E-commerce Landing Page

Product landing page with shopping cart functionality.

## Features

- Product showcase
- Shopping cart
- Responsive grid layout
- Add to cart functionality
- Price calculations

## Project Structure

```

│   └── products.css
├── js/

```

## Usage

Open `index.html` in browser.
