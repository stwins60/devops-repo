# Data Analysis with Pandas

Python application for data analysis using Pandas, NumPy, and Matplotlib.

## Features

- Data loading and cleaning
- Statistical analysis
- Data visualization
- CSV/Excel file processing
- Report generation

## Project Structure

```
04-data-analysis/
├── src/
│   ├── __init__.py
│   ├── analyzer.py
│   ├── visualizer.py
│   └── utils.py
├── data/
│   ├── sample_data.csv
│   └── .gitkeep
├── notebooks/
│   └── analysis.ipynb
├── output/
│   └── .gitkeep
├── requirements.txt
└── README.md
```

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python src/analyzer.py --input data/sample_data.csv --output output/
```

## Features

- Load CSV/Excel files
- Data cleaning and preprocessing
- Statistical analysis
- Generate visualizations
- Export reports
