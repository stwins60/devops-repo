# GraphQL API with Apollo

GraphQL server using Apollo Server.

## Features

- Apollo Server
- GraphQL schema
- Resolvers
- DataLoader
- GraphQL Playground

## Project Structure

```
02-graphql-apollo/
├── src/
│   ├── schema/
│   │   └── typeDefs.js
│   ├── resolvers/
│   │   └── index.js
│   ├── models/
│   │   └── User.js
│   └── server.js
├── package.json
└── README.md
```

## Installation

```bash
npm install
```

## Running

```bash
npm start
```

Access GraphQL Playground: http://localhost:4000
