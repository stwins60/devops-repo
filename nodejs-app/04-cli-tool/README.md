# CLI Tool with Commander

Command-line interface using Commander.js.

## Features

- CLI commands
- Interactive prompts
- Configuration management
- Progress indicators

## Project Structure

```
04-cli-tool/
├── bin/
│   └── cli.js
├── commands/
│   ├── user.js
│   └── config.js
├── package.json
└── README.md
```

## Installation

```bash
npm install
npm link
```

## Usage

```bash
mycli --help
mycli user list
mycli user add --name "John"
```
