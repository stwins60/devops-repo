# Real-time Chat with Socket.IO

WebSocket chat application using Socket.IO.

## Features

- Real-time messaging
- Socket.IO
- Room support
- User presence
- Message broadcasting

## Project Structure

```
03-socketio-chat/
├── src/
│   ├── server.js
│   ├── socket/
│   │   └── chatHandler.js
│   └── public/
│       ├── index.html
│       └── client.js
├── package.json
└── README.md
```

## Installation

```bash
npm install
```

## Running

```bash
npm start
```

Access at: http://localhost:3000
