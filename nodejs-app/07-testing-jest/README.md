# Testing with Jest

Comprehensive testing suite with Jest.

## Features

- Unit tests
- Integration tests
- Mocking
- Coverage reports
- Snapshot testing

## Project Structure

```
07-testing-jest/
├── src/
│   ├── calculator.js
│   ├── userService.js
│   └── utils.js
├── tests/
│   ├── calculator.test.js
│   ├── userService.test.js
│   └── utils.test.js
├── package.json
└── README.md
```

## Installation

```bash
npm install
```

## Running Tests

```bash
npm test
# With coverage
npm run test:coverage
# Watch mode
npm run test:watch
```
