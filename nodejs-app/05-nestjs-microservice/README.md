# Microservice with NestJS

Microservice using NestJS framework.

## Features

- NestJS framework
- Dependency injection
- Decorators
- Module system
- Testing support

## Project Structure

```
05-nestjs-microservice/
├── src/
│   ├── app.module.ts
│   ├── main.ts
│   ├── users/
│   │   ├── users.module.ts
│   │   ├── users.controller.ts
│   │   └── users.service.ts
│   └── common/
│       └── filters/
├── test/
├── package.json
└── README.md
```

## Installation

```bash
npm install
```

## Running

```bash
npm run start
# Development mode
npm run start:dev
```
